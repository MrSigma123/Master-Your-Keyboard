OVERVIEW:

Master-Your-Keyboard is the console program allowing the user to improve
typing skills. The program uses several sample text files, which are used
to compare the user input to the stream of the characters present in sample
files.

FUNCTIONALITIES:

- allowing the user to choose between several text files for practise
- hiding the text after typing the character
- coloring correct and innorect characters after submitting the line
- coloring exceeded or missed characters after submitting the line
- calculating the time counted from vieving the first line to ESC sequence
- showing the final summary message with all stats (all kinds of errors and time)
